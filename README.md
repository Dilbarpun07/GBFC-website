📦 Installation & Setup
Clone the repository:
git clone https://github.com/Dilbarpun07/GBFC-website.git
cd GBFC-website


Install dependencies:
npm install


Run the development server:
npm run dev


Build for production:
npm run build


Preview production build:
npm run preview
